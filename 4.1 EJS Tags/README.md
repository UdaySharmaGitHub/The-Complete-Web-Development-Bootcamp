# EJS Tags

| EJS Tags  | Description |
| :-------- | :------- |
| `<%= variable %>`      | `Js Ouput` | 
| `<% <% console.log("Hello") %> %>`      | `Js Execute` | 
| `<%- <h1>Hello</h2> %>`      | `Render HTML` | 
| `<% % % %>`      | `Show <% or %>` | 
| `<%# Comments %>`      | `Stop Execution` | 
| `<%- include("Header.ejs") %>`      | `Insert another EJS File` |
